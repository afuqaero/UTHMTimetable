# UTHM Timetable Planner

## About
Created by **feeq**, a final-year Bachelor of Software Engineering student at UTHM (Universiti Tun Hussein Onn Malaysia).

The aim of this simple mini-project is to allow ease of timetable arrangement for all semester students. 

## The Problem
When creating a timetable, I realized as a student that existing platforms don't feature an intuitive drag-and-drop functionality. This slows down the ability to quickly visualize and determine which subjects are available to add immediately. 

While I usually resorted to using a manual timetable builder, those tools often require paid services, require downloads, and still lack basic drag-and-drop support. This app was built to solve that!

## Usage Recommendation
It is highly recommended to use a **Desktop PC** for the full, optimal drag-and-drop arranging experience.
